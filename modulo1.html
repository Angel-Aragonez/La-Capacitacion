Submódulos

// SUB 1.1

Documentos Digitales

Se estudian las herramientas necesarias para crear, editar y dar formato a documentos de texto profesionales. El alumno aprende a estructurar información mediante estilos, tablas, índices y referencias, así como a compartir y exportar archivos en distintos formatos digitales.

// SUB 1.2

Aplicaciones Informáticas

Se hace uso de programas como MIT App Inventor para el desarrollo de aplicaciones móviles sencillas. El alumno aprende a diseñar interfaces, programar lógica básica y publicar aplicaciones en plataformas digitales, fomentando la creatividad y el pensamiento computacional.
